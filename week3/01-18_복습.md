#### map함수 쓰기

- `map(<func>,<list or tuple>)`에서 func의 매개변수가 두 개라면, list나 tuple을 넣을 때

  unpacking, `*list`와 같이 **iterable**하게 만들어서 넣어야한다!

  ```python
  number = [[1, 2, 3, 4, 5], [2, 2, 2, 2, 2]]
  def func1(x, y):
      return x**y
  
  lst = list(map(func1, *number))
  print(lst)
  ----------------------------------------------
  [1, 4, 9, 16, 25]
  ```

# 다형성

### 다형성(Polymorphism)

- 많은 형상을 가질 수 있는 성질
- 부모클래스의 참조변수로 자식클래스의 객체를 참조할 수 있다.



### 활용

```java
Parent P = new Child();
```

- 어차피 P는 객체의 주소값을 갖는 것이기 때문에, 자식의 주소값을 부모로 가질 수도 있다!
- 이 때 P.method 처럼 method를 부르면, 동적바인딩이 되서 자식꺼가 불러진다?
- 사실 뭔지 모르겠다.. 나중에 다시보자
